隐私政策

最后更新：2026年5月22日

Elori（"我们"、"我们的"或"我们"）致力于保护您的隐私。本隐私政策说明了 Elori 如何收集、使用和披露您的个人信息。本隐私政策适用于我们的网站及其相关子域名（统称为我们的"服务"）以及我们的应用程序 Elori。通过访问或使用我们的服务，即表示您已阅读、理解并同意我们按照本隐私政策和使用条款中所述的方式收集、存储、使用和披露您的个人信息。

1. 我们收集的信息

A. 您提供给我们的信息

当您执行以下操作时，我们可能会收集您自愿提供的个人信息：

  • 联系我们寻求支持或咨询问题。
  • 提供反馈或评价我们的应用。

B. 自动收集的信息

  • 设备信息：我们可能会收集有关您移动设备的信息，包括硬件型号、操作系统版本、唯一设备标识符和移动网络信息。
  • 诊断和崩溃数据：我们使用 Firebase Crashlytics（一项 Google 服务）来帮助我们了解应用崩溃、性能问题和稳定性问题。崩溃报告可能包括设备和操作系统信息、应用版本、时间戳、崩溃追踪、诊断标识符以及诊断问题所需的相关技术数据。
  • 使用和应用状态数据：我们可能会收集有限的应用活动和应用状态信息，以便提供、维护、保护和改进服务。

C. 内容、模板和 Firebase 服务

Elori 使用 Firebase 和 Google Cloud 服务来提供和运营应用的部分功能。这包括基于 Firestore 的配置或内容记录，以及基于 Firebase Storage 或 Google Cloud Storage 的资源，例如 Elori 内提供的模板、示例、图片或其他应用内容。这些服务可能会处理技术请求信息、设备或应用标识符、时间戳、IP 地址以及内容传输元数据，以便提供所请求的内容并确保应用可靠运行。

D. 本地缓存与 UserDefaults

Elori 会使用应用存储、缓存和 Apple UserDefaults 在你的设备上本地存储一些信息。这可能包括偏好设置、引导状态、订阅或购买状态、最近加载的内容元数据、缓存资源,以及帮助应用记住你的选择并更快运行的设置。本地数据会保留在你的设备上,除非你主动移除、被 iOS 清除,或在卸载应用时删除。

E. 相册访问与导出

为了提供应用的核心功能(编辑照片和制作轮播图),Elori 可能会请求访问你选择的照片,以及将完成的图片保存或导出到设备相册的权限。

  • 本地处理:我们非常重视你的隐私。所有照片处理和编辑都设计为在你的设备上本地完成。我们不会使用你的个人照片来训练模型,也不会查看你的相册。
  • 导出权限:如果你选择保存或导出作品,iOS 可能会请求将导出的图片添加到相册的权限。你可以随时在设备设置中撤销相册权限,但这可能会限制导入或导出功能。

2. 我们如何使用你的信息

我们使用收集的信息来:

  • 提供、运营和维护我们的应用。
  • 改进、个性化和扩展我们的应用。
  • 了解和分析你如何使用我们的应用。
  • 处理你的订阅交易(通过 Apple App Store 管理)。
  • 提供应用内容、模板、配置和相关资源。
  • 诊断崩溃、错误和性能问题。
  • 发现并防止欺诈行为。
  • 回复你的评论、问题,并提供客户服务。

3. 订阅与付款

我们的应用通过 Apple 的应用内购买系统提供付费订阅方案和终身买断选项。当前 StoreKit 产品可能包括：3 天免费试用后按周续订的周订阅、按年自动续订的年订阅，以及终身买断。

  • Apple App Store：所有付款通过 Apple App Store 处理。我们不存储或收集你的支付卡信息。这些信息直接提供给 Apple，其对你个人信息的使用受其隐私政策约束。

4. 第三方服务

我们使用第三方服务商来运营、交付、诊断和改进本服务。这些服务商仅在提供其服务所需的范围内为我们处理信息。

  • Firebase 及 Google 服务，包括 Firebase Crashlytics、Firestore 和 Firebase Storage 或 Google Cloud Storage，用于诊断、应用配置、数据库支持的内容及内容交付。
  • Apple App Store 和 StoreKit，用于购买、订阅、试用、收据、续订、取消和退款。

5. 数据保留

我们仅在本隐私政策所述目的所需的期限内保留信息，包括提供应用、维护内容交付、诊断崩溃、履行法律义务、解决争议和执行我们的政策。崩溃和诊断信息仅在可靠性、安全性和支持所需的期限内保留。Firebase 托管的应用内容及相关服务日志可能根据我们的运营需求和 Google Firebase 保留政策进行保留。本地缓存和 UserDefaults 保留在你的设备上，直到被应用、iOS 或你自己清除。

你可以通过 tryelori.com@gmail.com 联系我们，请求删除我们控制的个人信息。由于 Elori 不需要账户，我们可能需要你提供信息以定位与你请求相关的记录。购买、订阅、退款、取消记录和账单历史由 Apple 管理，可能需要通过你的 Apple ID 或 Apple 支持处理。

6. 你的选择

  • 你可以在 iOS 设置中管理相册权限。
  • 你可以通过删除应用来清除本地存储的数据。
  • 你可以在 Apple ID 订阅设置中管理或取消订阅。
  • 你可以联系我们请求访问、更正或删除我们掌握的个人信息。

7. 儿童隐私

我们不会有意收集 13 岁以下儿童的个人身份信息。如果你是家长或监护人，发现你的孩子向我们提供了个人数据，请联系我们。

8. 隐私政策更新

我们可能会不时更新隐私政策。我们会在此页面发布新的隐私政策来通知你任何变更。建议你定期查看本隐私政策以了解最新变化。
