常见问题

订阅

如何管理或取消我的订阅？

Elori 订阅通过你的 Apple ID 管理。打开 iPhone 上的"设置"应用，轻点你的名字，轻点"订阅"，然后选择 Elori 来查看、更改或取消你的套餐。

删除应用会取消订阅吗？

不会。从设备中删除 Elori 不会取消有效订阅。你需要在 Apple ID 订阅设置中取消。

如何恢复购买？

确保你使用了当初购买时的同一 Apple ID 登录，然后在 Elori 中使用恢复购买选项。如果访问权限仍未更新，请重启应用并检查你的 Apple ID 订阅状态。

退款

如何申请退款？

Apple 处理所有 App Store 购买和退款。要申请退款，请访问 reportaproblem.apple.com，使用你的 Apple ID 登录，选择 Elori，然后按照 Apple 的退款申请步骤操作。

Elori 可以直接退款吗？

不可以。因为账单由 Apple 处理，退款必须通过 Apple 申请。我们很乐意帮你了解去哪里操作或解答关于应用的问题，但我们无法自己批准或处理 App Store 退款。

支持

如何联系 Elori？

请发送邮件至 tryelori.com@gmail.com。请附上你使用的设备、iOS 版本以及简短描述发生的情况，以便我们更快地帮助你。

购买后如果 Pro 功能不可用该怎么办？

首先，确认购买或订阅在你的 Apple ID 订阅中处于激活状态。然后重新打开 Elori，尝试在应用内恢复购买。如果仍然无法解决，请联系我们并提供详细信息，我们会帮助你排查问题。

隐私与照片

使用 Elori 需要注册账号吗？

不需要。Elori 无需注册或登录即可使用。

Elori 会存储我的照片吗？

Elori 会在你的设备上处理照片。我们不会在服务器上存储、托管或查看你的照片。
